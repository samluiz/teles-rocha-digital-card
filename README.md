# Teles Rocha - Digital Card

Single-page mini portfolio/business card for a musician.

## Run locally

Open `index.html` in your browser.

## Deploy on GitHub Pages

1. Create a GitHub repository and push this folder contents to the default branch (`main`).
2. On GitHub, go to **Settings > Pages**.
3. In **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
4. Save and wait for the deployment.
5. Your site will be available at:
   `https://<your-username>.github.io/<repository-name>/`

## Notes

- The QR code is generated dynamically to match the current page URL.
- Update contact links in `index.html` if needed.
